### Landsat engine (`ndvi/engines/landsat.py`)

STAC-based rewrite. Defaults to Microsoft Planetary Computer (`landsat-8-c2-l2`), bands `B4`/`B5` (no SCL). Configurable via `NDVI_LANDSAT_*` env vars.

### MODIS engine (`ndvi/engines/modis.py`)

STAC-based rewrite with pre-computed NDVI band. Defaults to Microsoft Planetary Computer (`modis-13q1-061`), band `NDVI` (scale 0.0001), `DetailedQA` mask. Uses GDAL/rasterio for COG download and band extraction.

### GEE engine (`ndvi/engines/gee.py`)

STAC-based rewrite (independently configurable from `stac` engine). Defaults to Copernicus CDSE (`sentinel-2-l2a`), bands `B04_10m`/`B08_10m`/`SCL`. Configurable via `NDVI_GEE_*` env vars.

### `@extend_schema(auth=...)` fix

`AlertDetailView` and `PodcastRefreshView` had `auth=` at the class level, which bled into every method's OpenAPI schema. Moved to the specific method(s) that need it; added `request=None` on idempotent PATCH/POST endpoints.
