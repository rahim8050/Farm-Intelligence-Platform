# Daily Report - 2026-06-04

## Summary

Shipped radio Phase 3: per-user favorites and per-user listening history,
both auth'd (JWT or API key) and scoped to the calling user. `StationStreamView`
now auto-records a `ListeningHistory` row whenever an authenticated client
opens a stream. All four new endpoints (plus the stream auto-record) are
documented end-to-end (module / class / method docstrings, `@extend_schema`
envelope, regenerated `schema.yml`).
