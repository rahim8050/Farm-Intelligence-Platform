# Daily Report — 2026-04-04 (EAT)

## Scope of Work
- Investigated why healthy farms were generating solid green PNGs.
- Diagnosed the NDVI-to-PNG normalization issue.
- Implemented histogram normalization as the new default.
- Added tests and verified quality checks.

## Root Cause
- NDVI was calculated correctly.
- The old normalization formula compressed values into the green part of the colormap.
- Result: the PNGs looked uniformly green.

## Solution
- Added histogram normalization and kept fixed normalization as a legacy option.
- Updated the PNG rendering path to accept the normalization mode.
- Added a settings helper to choose the default normalization.

## Configuration
- Default: `histogram`
- Optional legacy mode: `NDVI_COLORMAP_NORMALIZATION=fixed`
- Restart required: Django and Celery after deployment

## Assessment
- NDVI computation was correct; visualization normalization was the issue.
- Histogram mode provides maximum detail within each farm.
- The change is backward compatible.
